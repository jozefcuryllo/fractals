FRACTALS
FOR x86_64 LINUX + SDL2

[DESCRIPTION]
A real-time fractal(-ish) generator written in C++ with SDL2. 
Shows various fractals and non-fractal animations that
just look cool.

[SYSTEM REQUIREMENTS]
  - OS: Linux (x86_64)
  - Dependencies: SDL2 with TTF, Glibc
  - Tools: GCC, cmake

[BUILD INSTRUCTIONS]
make release
./build/Fractal
